## 1. Operations Branding

### 1.1 Phlebotomist Uniform

Uniforms should make collection partners look professional, identifiable, and trustworthy.

#### Polo T-Shirt Rules

| Element | Recommendation |
|---|---|
| Base Colour | Black, white, or amber/orange depending on role and season. |
| Logo Placement | Left chest. |
| Wordmark | Use horizontal lockup if space allows. |
| Back Print | Optional: “Diagnostics, democratized.” or “Zelnoo Collection Partner”. |
| Fabric | Summer-friendly, breathable, anti-wicking/quick-dry preferred for field work. |
| Buttons / Accents | Zelnoo orange or black. |

#### Preferred Uniform Combination

- Black polo with orange logo for premium daily use.
- White polo with orange logo for summer or clinical contexts.
- Black cap with orange logo.
- Black backpack/bag with orange logo.
- ID card with photo, QR/ID, and emergency contact.

### 1.2 Phlebotomist Bag Branding

The phlebotomist bag is a high-visibility brand touchpoint.

#### Bag Branding Rules

| Element | Recommendation |
|---|---|
| Logo | Use icon + ZELNOO wordmark. |
| Tagline | “Diagnostics, democratized.” or service-specific claim. |
| Service Highlight | “30 mins sample collection*” only if operationally accurate in the service area. |
| QR Code | Use Zelnoo orange only if contrast remains scannable; otherwise black QR with orange frame. |
| Print Location | Front panel, away from seams, straps, curves, and zippers. |
| Print Method | Screen print, DTF, heat transfer, UV print, or embroidery patch depending on material. |

#### Bag Copy Options

- Diagnostics, democratized.
- Doorstep sample collection.
- Trusted diagnostic pickup.
- Scan to book tests.
- Sample collection partner.

### 1.3 ID Cards

ID cards must prioritize safety, trust, and verification.

#### Front Side

- Zelnoo logo.
- Employee/partner photo.
- Full name.
- ID number.
- Role: Collection Partner / Phlebotomist / Operations.
- QR code for verification.
- Validity date.

#### Back Side

- Company name.
- Emergency contact.
- Instructions: “If found, please return to Zelnoo Private Limited.”
- Disclaimer: “This ID is company property.”

### 1.4 Sample Collection Labels

Sample labels must not be treated as marketing assets. They are operational and must prioritize readability.

Use:

- Black text on white.
- Barcode/QR code.
- Sample ID in DM Mono.
- Patient initials or anonymized ID where required.
- Collection date/time.
- Test/vial type.
- Lab destination.

Avoid:

- Orange backgrounds behind barcode.
- Decorative graphics.
- Low-contrast text.
- Overcrowded labels.

### 1.5 Inventory and Vendor PO Templates

Use a consistent document structure:

- Zelnoo logo top left.
- Document title top right.
- Orange divider line.
- Vendor details.
- PO number in DM Mono.
- Table with SKU, item, category, unit, quantity, rate, GST, total.
- Footer with authorized signatory.
- Notes in neutral grey.

## 2. Corporate Templates

### 2.1 Business Card

#### Front

- Zelnoo logo centered or top left.
- Black or white premium background.
- Optional tagline.

#### Back

- Name.
- Designation.
- Phone.
- Email.
- Website.
- Address if required.
- QR code optional.

#### Recommended Finish

- Matte lamination.
- Spot UV on logo or wordmark.
- Orange edge paint optional for premium cards.
- Avoid glossy overcrowded designs.

### 2.2 Letterhead

| Element | Rule |
|---|---|
| Header | Logo top left, document metadata optional top right. |
| Accent | Thin orange line below header. |
| Body | DM Sans / Aptos, black text. |
| Footer | Company legal name, CIN/GST/address/contact if applicable. |
| Signature | Black text, orange signatory line optional. |

### 2.3 Email Signature

#### Standard Format

```text
Name Surname
Designation | Zelnoo Private Limited
M: +91 XXXXX XXXXX | E: name@zelnoo.com
W: www.zelnoo.com
Diagnostics, democratized.
```

#### Signature Rules

- Keep logo width around 120–150 px.
- Avoid large banners unless part of a campaign.
- Avoid multiple colours in text.
- Use black, grey, and orange only.
- Avoid motivational quotes in official email signatures.

### 2.4 Invoice / Receipt / Payment Confirmation

Use minimal branding. Prioritize clarity and compliance.

- Logo top left.
- Document type top right.
- Orange divider.
- Customer / vendor details.
- Transaction ID in DM Mono.
- Amount and GST clearly separated.
- Payment status chip.
- Footer with legal and support details.

## 4. Print and Production Guidelines

### 4.1 Print Colours

For print, request colour matching against the primary HEX `#F08E04` and gradient artwork. Vendors may ask for CMYK/Pantone approximations; final approval should be based on printed proof.

Approximate CMYK starting point for `#F08E04`:

```text
C: 0
M: 41
Y: 98
K: 6
```

This is only a production starting point. Always approve physical samples for uniforms, bags, cards, signage, and stickers.

### 4.2 Printing Methods

| Asset | Recommended Method |
|---|---|
| Business cards | Offset/digital print, matte lamination, spot UV optional. |
| Uniform polo | Embroidery or DTF/heat transfer depending on fabric. |
| Cap | Embroidery. |
| Phlebotomist bag | Screen print, DTF, UV print, or stitched patch. |
| ID card | PVC print with lamination. |
| Stickers | Vinyl waterproof print. |
| Letterhead | Digital/offset print or MS Word template. |
| Posters | Digital print with matte finish. |

### 4.3 Vendor Handoff Checklist

Before sending files to a vendor, include:

- Logo in SVG/PDF/CDR/AI where possible.
- PNG preview.
- Exact print size in mm.
- Placement mockup.
- Colour code `#F08E04`.
- Material/fabric details.
- Print method preference.
- Quantity.
- Approval requirement for sample/proof.
- Whether background should be transparent.

## 7. Department-Specific Brand Rules

### 7.1 Marketing

- Use only approved logos, fonts, colours, and templates.
- Maintain brand ratio: white/black dominance with controlled orange.
- Avoid random templates from Canva that dilute the brand.
- Keep offer creatives legally and operationally accurate.
- Every external creative should include logo, CTA, and disclaimer where required.

### 7.2 Sales

- Use approved pitch deck template.
- Use clean diagrams and consistent icons.
- Use brand orange for key numbers and CTA.
- Do not use unverified claims in sales material.
- Use “diagnostics marketplace/platform” consistently.

### 7.3 HR

- Use formal templates for offer letters, appointment letters, ID cards, onboarding, policies, and notices.
- Keep visual branding minimal and professional.
- Use Zelnoo Private Limited in legal employment documents.
- Use department colour only for internal visual navigation, not legal identity.

### 7.4 Finance and Accounts

- Use clean, compliant templates.
- Use monochrome or minimal orange.
- Avoid decorative backgrounds on invoices, POs, receipts, and salary documents.
- Use consistent document numbers and entity IDs.

### 7.5 Operations

- Uniforms, bags, ID cards, labels, and vendor templates must use approved logo files.
- Use high contrast for field assets.
- QR codes must be tested after printing.
- Do not modify logo artwork for stitching or printing without approval.

### 7.6 Legal and Compliance

- Use Zelnoo Private Limited where legal entity name is required.
- Avoid marketing phrases in legal agreements.
- Use black text on white background.
- Logo can appear in header but should not dominate legal documents.

## 11. Brand Incident Handling

A brand incident happens when incorrect or outdated branding is used publicly or sent to vendors/customers.

### 11.1 Examples

- Old colour `#F5B400` used in a new public creative.
- Logo stretched on a banner.
- Incorrect wordmark spelling.
- Unsupported medical claim in social post.
- Wrong QR code on bag/poster.
- Low-quality screenshot logo used in print.
- Vendor prints wrong colour on uniform/bag.

### 11.2 Correction Process

1. Identify asset and channel.
2. Stop further distribution if possible.
3. Replace with approved version.
4. Notify vendor/team if physical production is involved.
5. Update source file in central folder.
6. Record issue in brand audit log.
7. Add prevention rule if repeated.

### 11.3 Brand Audit Log Format

| Date | Asset | Issue | Owner | Correction | Status |
|---|---|---|---|---|---|
| DD/MM/YYYY | Example: Bag print file | Wrong orange shade | Ops | Replaced with approved PDF | Closed |

## 14. Final Implementation Principle

The brand is not only a logo or colour. For Zelnoo, the brand must show up in the way a customer books a test, the way a phlebotomist collects a sample, the way a lab uploads a report, the way a doctor accesses patient information, the way finance sends a PO, and the way sales presents the platform.

The system must remain consistent across every surface:

```text
Same logo.
Same orange.
Same typography.
Same trust-first language.
Same operational clarity.
Same premium healthcare technology feel.
```

That is the standard for Zelnoo.
